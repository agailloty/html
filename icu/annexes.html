addPrefix <- function(x, prefix) {
  replacements <- list(
    espace = c(" ", "_"),
    virgule = c(",", "_ou"),
    trait = c("-","_"),
    slash = c("/", "_"),
    angers = c("d'Angers", ""),
    e = c("é", "e")
  )
  
  for (repl in replacements) {
    new <- gsub(pattern = repl[1], replacement = repl[2], x)
    x <- new
  }
  return (paste0(prefix, toupper(x)))
}

createLowerInterval <- function(x) {
  dplyr::case_when(
    x <= 5 ~ 0,
    x > 5 & x <= 10 ~ 5,
    x > 10 & x <= 20 ~ 10,
    x > 20 & x <= 30 ~ 20,
    x > 30 ~ 30
  )
}

createUpperInterval <- function(x) {
  dplyr::case_when(
    x <= 5 ~ 5,
    x > 5 & x <= 10 ~ 10,
    x > 10 & x <= 20 ~ 20,
    x > 20 & x <= 30 ~ 30,
    x > 30 ~ 50
  )
}

11.2 Diffusion du questionnaire

Le questionnaire a été principalement diffusé dans un groupe Facebook appelé Tu sais que tu es d’Angers quand…*. Ce groupe compte plus de 30K membres et est représentative de la population d’Angers et du Maine-et-Loire.

11.2.1 Retours des participants

Dans le groupe où le questionnaire a été diffusé plusieurs retours m’ont été faits par les personnes qui ont répondu au questionnaire. Globalement tous les retours ont été positifs ; beaucoup de répondants sont ravis que l’étude s’intéresse à cette question de chaleur urbaine.
J’ai également reçu quelques critiques concernant le mode de paiement que j’ai choisi qui est la taxe d’habitation du fait que la taxe d’habitation va être supprimée d’ici quelques années.
